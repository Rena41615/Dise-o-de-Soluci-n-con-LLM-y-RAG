# Asistente Virtual - Metro de Santiago (Evaluación Parcial 1)

Este repositorio contiene la primera etapa de una solución basada en LLM para asistir a los pasajeros del Metro de Santiago, cumpliendo con el indicador de logro IL1.1.

## Requisitos Previos
- Python 3.10 o superior.
- Credenciales de GitHub Models (Token PAT).

## Instrucciones de Ejecución
1. Clonar el repositorio.
2. Crear un entorno virtual (`python -m venv .venv`) y activarlo.
3. Instalar las dependencias necesarias (`pip install openai python-dotenv`).
4. Crear un archivo `.env` en la raíz del proyecto basándose en las variables necesarias (GITHUB_BASE_URL y GITHUB_TOKEN).
5. Ejecutar el archivo `EP1_Metro.ipynb` utilizando Jupyter Lab o Visual Studio Code.
